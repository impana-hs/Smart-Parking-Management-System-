# Smart-Parking-Management-System-
Smart Parking Management System using Arduino UNO, IR Sensors, Servo Motor, and LCD for real-time parking slot monitoring and automated gate control.

<h2>📌 Project Overview</h2>

<p>
The <b>Smart Parking Management System</b> is an embedded automation project designed to efficiently manage parking spaces using <b>Arduino UNO</b>, <b>IR Sensors</b>, <b>Servo Motors</b>, and an <b>I2C LCD Display</b>. The system automatically detects vehicle entry and exit, updates parking slot availability in real time, and controls gate operations without manual intervention.
</p>

<p>
This solution helps reduce traffic congestion within parking areas, optimizes parking space utilization, and enhances user convenience through automated monitoring and management.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
  <li>Real-time parking slot monitoring.</li>
  <li>Automatic vehicle entry and exit detection using IR sensors.</li>
  <li>Servo motor-based gate automation.</li>
  <li>Live parking slot availability displayed on LCD.</li>
  <li>Reduced manual effort through automation.</li>
  <li>Efficient parking space utilization.</li>
  <li>Low-cost and scalable parking management solution.</li>
</ul>

<h2>⚙️ System Workflow</h2>

<ol>
  <li>Vehicle approaches the parking entrance.</li>
  <li>IR sensor detects vehicle presence.</li>
  <li>Available parking slots are verified.</li>
  <li>Servo motor automatically opens the gate.</li>
  <li>Vehicle enters the parking area.</li>
  <li>Parking count is updated in real time.</li>
  <li>LCD displays available parking slots.</li>
  <li>Exit sensor detects vehicle departure.</li>
  <li>Parking slot count is updated automatically.</li>
</ol>
